Um conceito utilizando parte do projeto FEGUI ( Fast Engine Graphical User Interface ) vai trazer novos recursos para o jogo.
Janelas com viewports podem ser abertas simultaneamente.
Depurador de núcleo poderá resolver problemas com erros ocorridos durante a execução do programa.
O relatório de erros mostra detalhes da exceção ocorrida durante a violação de acesso ou outro tipo de erro causado.
O console do jogo será nada mais do que um shell ( Terminal ), onde digita os comandos.
O HUD ( Head Up Display ) será baseado em widgets, formando janelas de fundos do jogo. Irá ficar visivel sobre a janela do jogo.

Janelas semi-transparentes com cores (ciano/azul claro), amarelas, verdes.
Estas janelas são HUDs baseados em widgets.
Também podem conter três corês quando é usado em caso de bandeiras, avisos e estados de configurações.
Neste caso, textos e logotipos usam corês de acordo com o resultado:
Verde é para resultado positivo (OK)
Amarelo é para resultado de aviso (Warning)
Vermelho é para resultado de erro (Error)
As janelas semi-transparentes ficam sem borda e podem ser arrastadas como estivessem no fundo.
